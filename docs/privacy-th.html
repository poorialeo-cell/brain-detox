นโยบายความเป็นส่วนตัว

ปรับปรุงล่าสุด: 24 พฤษภาคม 2026

日本語 English ภาษาไทย

brain-detox (ต่อไปนี้เรียกว่า "แอป") เคารพความเป็นส่วนตัวของผู้ใช้ และจัดเก็บข้อมูลส่วนบุคคลให้น้อยที่สุดเท่าที่จำเป็น นโยบายนี้อธิบายประเภทของข้อมูลที่แอปจัดการ วัตถุประสงค์ สถานที่จัดเก็บ และสิทธิ์ของผู้ใช้

1. ข้อมูลที่เราเก็บ

1-1. รหัสประจำตัวแบบไม่ระบุตัวตน

แอปใช้ Firebase Authentication แบบ Anonymous Sign-In เพื่อออกรหัสประจำตัว (UID) ที่ไม่ระบุตัวตนสำหรับผู้ใช้แต่ละราย รหัสนี้ ไม่เชื่อมโยงกับชื่อ อีเมล หมายเลขโทรศัพท์ หรือข้อมูลส่วนบุคคลใด ๆ

1-2. ข้อมูลภายในแอป

ข้อมูลต่อไปนี้จะถูกจัดเก็บที่เซิร์ฟเวอร์ของเรา (Google Cloud Firestore, ภูมิภาคโตเกียว asia-northeast1):

ข้อมูลรายละเอียดวัตถุประสงค์
การตั้งค่าพาร์ทเนอร์ประเภทพาร์ทเนอร์ที่เลือกปรับข้อเสนอกิจกรรม
คะแนนสมองค่าตัวเลข 0–100ฟังก์ชันหลักของแอป
ภาษาja / en / thการแสดงผล UI
ประวัติคะแนนบันทึก (สูงสุด 60 รายการ)แสดงประวัติ คำนวณสตรีค
การตั้งค่าการแจ้งเตือนเปิด/ปิด เวลาเตือนเฉพาะบนอุปกรณ์
สถานะเหรียญตราเฉพาะบนอุปกรณ์แสดงความสำเร็จ
ตัวนับ Rate Limitจำนวนการเรียก API ต่อวันป้องกัน ควบคุมต้นทุน

1-3. สิทธิ์อุปกรณ์

แอปใช้เฉพาะ การแจ้งเตือน (Push, เลือกได้) เท่านั้น ไม่เข้าถึง ตำแหน่งที่ตั้ง รายชื่อผู้ติดต่อ กล้อง ไมโครโฟน คลังภาพ ข้อมูลสุขภาพ หรือรหัสโฆษณา (IDFA / AAID)

2. บริการบุคคลที่สาม

2-1. Firebase (Google LLC)

แอปใช้บริการของ Firebase (Authentication / Firestore / Functions / App Check)

นโยบายของ Firebase: https://firebase.google.com/support/privacy

2-2. OpenAI (OpenAI, L.L.C.)

เพื่อสร้างข้อเสนอกิจกรรม ข้อมูลต่อไปนี้จะถูกส่งไปยัง OpenAI API (เซิร์ฟเวอร์สหรัฐฯ):

ส่งไปไม่ส่งไป
คะแนนสมองปัจจุบัน (0–100)UID ไม่ระบุตัวตน
ประเภทพาร์ทเนอร์ที่เลือกข้อมูลอุปกรณ์
ภาษาข้อมูลที่ระบุตัวบุคคลได้
สรุปกิจกรรมล่าสุด (สูงสุด 3 ชื่อ)ประวัติคะแนนละเอียด

ตามนโยบายของ OpenAI ข้อมูลที่ส่งผ่าน API จะไม่ถูกใช้ในการเทรนโมเดล

นโยบายของ OpenAI: https://platform.openai.com/docs/models/how-we-use-your-data

2-3. Expo (Expo, Inc.)

แอปสร้างด้วย Expo framework แต่ ไม่มีการส่งข้อมูลผู้ใช้ไปยังเซิร์ฟเวอร์ของ Expo

3. การเก็บรักษาและการลบข้อมูล

3-1. ระยะเวลาการเก็บรักษา

หลังจากที่คุณถอนการติดตั้งแอป ข้อมูลใน Firestore จะยังคงอยู่ เพื่อให้สามารถกู้คืน UID เดิมเมื่อติดตั้งใหม่

3-2. วิธีการลบ

  1. ในแอป: ตั้งค่า → รีเซ็ตข้อมูล
    ลบโปรไฟล์และประวัติคะแนนใน Firestore ล้างการตั้งค่าทั้งหมดบนอุปกรณ์
  2. ติดต่อเรา
    ส่งอีเมลตามที่อยู่ด้านล่าง ระบุ UID (ดูได้ในหน้าตั้งค่า) เพื่อความรวดเร็ว

4. การเปิดเผยต่อบุคคลที่สาม

เราจะไม่เปิดเผยหรือขายข้อมูลผู้ใช้ให้กับบุคคลที่สาม ยกเว้นเมื่อกฎหมายกำหนด

5. ความเป็นส่วนตัวของเด็ก

แอปนี้ ไม่เหมาะสำหรับผู้ใช้อายุต่ำกว่า 13 ปี หากผู้ปกครองพบว่าบุตรหลานที่อายุต่ำกว่า 13 ปีใช้แอปนี้ กรุณาลบข้อมูลด้วยวิธีข้างต้น

6. สิทธิ์ของผู้ใช้

ภายใต้ GDPR, CCPA และกฎหมายที่บังคับใช้อื่น ๆ ผู้ใช้มีสิทธิ์:

  • สิทธิ์เข้าถึงข้อมูลของตนเอง
  • สิทธิ์ขอแก้ไข
  • สิทธิ์ขอลบ ("สิทธิ์ที่จะถูกลืม")
  • สิทธิ์คัดค้านการประมวลผล
  • สิทธิ์ในการเคลื่อนย้ายข้อมูล

7. ความปลอดภัย

  • การสื่อสารทั้งหมดเข้ารหัสด้วย HTTPS / TLS
  • Firestore Rules ทำให้ผู้ใช้แต่ละรายเข้าถึงได้เฉพาะข้อมูลของตน
  • คีย์ OpenAI API เก็บไว้ใน Firebase Functions Secret Manager และไม่เปิดเผยต่อไคลเอนต์
  • Firebase App Check รับคำขอจากแอปที่ถูกต้องเท่านั้น

8. คุกกี้และการติดตาม

แอปนี้เป็นแอปมือถือ ไม่ใช้คุกกี้ ไม่ใช้รหัสโฆษณาหรือรหัสติดตาม

9. การเปลี่ยนแปลงนโยบาย

นโยบายนี้อาจมีการเปลี่ยนแปลงโดยไม่แจ้งให้ทราบล่วงหน้า การเปลี่ยนแปลงที่สำคัญจะประกาศผ่านการแจ้งเตือนในแอปหรือบันทึกการอัปเดต
