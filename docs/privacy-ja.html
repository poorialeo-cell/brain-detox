プライバシーポリシー

最終更新日: 2026年5月24日

日本語 English ภาษาไทย

brain-detox（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人を特定できる情報を最小限の収集にとどめます。本ポリシーは、本アプリが取り扱うデータの種類、目的、保存場所、ユーザーの権利を説明します。

1. 収集するデータ

1-1. 匿名識別子

本アプリは Firebase Authentication の匿名サインイン を利用して、ユーザーごとに一意の匿名 ID（UID）を発行します。この ID は氏名・メールアドレス・電話番号などの個人情報とは紐付きません。

1-2. アプリ内データ

以下のデータをサーバー側（Google Cloud Firestore / 東京リージョン asia-northeast1）に保存します。

データ内容目的
パートナー設定選択中のパートナータイプアクション提案のパーソナライズ
ブレインスコア0〜100 の数値アプリ機能の中核
言語設定ja / en / thUI 表示
スコア履歴アクション・テストの記録（最大60件）履歴表示・ストリーク計算
通知設定有効/無効、リマインド時間端末ローカルのみ
バッジ取得状況端末ローカルのみ実績表示
レート制限カウンタ日次集計サービス保護・コスト管理

1-3. デバイス権限

本アプリが使用する権限は 通知（プッシュ通知のみ・任意） です。位置情報、連絡先、カメラ、マイク、写真ライブラリ、健康データ、広告 ID（IDFA / AAID）には一切アクセスしません。

2. 第三者サービスへのデータ送信

2-1. Firebase（Google LLC）

本アプリは Google が提供する Firebase（Authentication / Firestore / Functions / App Check）を利用します。

Firebase のプライバシーポリシー: https://firebase.google.com/support/privacy

2-2. OpenAI（OpenAI, L.L.C.）

アクション提案・パートナーメッセージ生成のため、以下を OpenAI API（米国サーバー） に送信します。

送信されるもの送信されないもの
現在のブレインスコア（0〜100）匿名 UID
選択中のパートナータイプ端末情報
言語設定個人を特定できる情報
直近のアクション履歴（最大3件のタイトル要約）スコア履歴の詳細

OpenAI に送信したデータは、OpenAI のポリシーに従い モデル学習には使用されません。

OpenAI の API データ利用ポリシー: https://platform.openai.com/docs/models/how-we-use-your-data

2-3. Expo（Expo, Inc.）

本アプリは Expo フレームワークでビルドされていますが、ユーザーデータを Expo のサーバーに送信することはありません。

3. データの保存期間と削除

3-1. 保存期間

ユーザーが本アプリをアンインストールした後も、Firestore に保存されたデータは残り続けます。これは、再インストール時に同一の匿名 UID でデータを復元するためです。

3-2. 削除方法

  1. アプリ内: 設定 → データをリセット
    Firestore に保存されたプロフィール・スコア履歴がすべて削除されます。端末ローカルの設定もすべて消去されます。
  2. 問い合わせ
    下記の連絡先までメールにて削除依頼をしてください。匿名 UID（設定画面で確認可能）を添えると確実です。

4. データの開示・第三者提供

本アプリは、法令に基づく場合を除き、ユーザーのデータを第三者に開示・販売することはありません。

5. 子どものプライバシー

本アプリは 13歳未満のお子様による利用を想定していません。13歳未満のお子様の保護者の方で、お子様が本アプリを利用していることに気づかれた場合は、上記の削除方法でデータを削除してください。

6. ユーザーの権利

GDPR、CCPA、その他適用される法令に基づき、以下の権利を有します。

  • 保存されている自分のデータへのアクセス権
  • データの訂正を要求する権利
  • データの削除を要求する権利（「忘れられる権利」）
  • データ処理に異議を申し立てる権利
  • データポータビリティの権利

7. セキュリティ

  • 通信はすべて HTTPS / TLS で暗号化 されます
  • Firestore Rules により、ユーザーは自分のデータのみアクセス可能です
  • OpenAI API キーは Firebase Functions の Secret Manager で管理され、クライアントに公開されません
  • Firebase App Check により、正規のアプリからのリクエストのみ受け付けます

8. クッキー・トラッキング

本アプリはモバイルアプリのため クッキーは使用しません。広告 ID やトラッキング ID も使用しません。

9. ポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合は、アプリ内通知またはアプリ更新時のお知らせでご案内します。
