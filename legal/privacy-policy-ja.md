# プライバシーポリシー

**最終更新日: 2026年5月24日**

brain-detox（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人を特定できる情報を最小限の収集にとどめます。本ポリシーは、本アプリが取り扱うデータの種類、目的、保存場所、ユーザーの権利を説明します。

---

## 1. 収集するデータ

### 1-1. 匿名識別子

本アプリは **Firebase Authentication の匿名サインイン** を利用して、ユーザーごとに一意の匿名 ID（UID）を発行します。この ID は氏名・メールアドレス・電話番号などの個人情報とは紐付きません。

### 1-2. アプリ内データ

以下のデータをサーバー側（Google Cloud Firestore / 東京リージョン `asia-northeast1`）に保存します。


| データ       | 内容                                                      | 目的                       |
| --------- | ------------------------------------------------------- | ------------------------ |
| パートナー設定   | 選択中のパートナータイプ（teacher / counselor / scientist / trainer） | アクション提案のパーソナライズ          |
| ブレインスコア   | 0〜100 の数値                                               | アプリ機能の中核（脳の状態指標）         |
| 言語設定      | `ja` / `en` / `th`                                      | UI 表示                    |
| スコア履歴     | アクション完了・ブレインロットテストの記録（最大60件）                            | 履歴表示・ストリーク計算             |
| 通知設定      | 通知の有効/無効、リマインド時間                                        | 端末ローカルに保存（クラウドには送信されません） |
| バッジ取得状況   | 端末ローカルに保存（クラウドには送信されません）                                | 実績表示                     |
| レート制限カウンタ | API 呼び出し回数の日次集計                                         | サービス保護・コスト管理             |


### 1-3. デバイス権限


| 権限         | 用途                | 任意 / 必須                |
| ---------- | ----------------- | ---------------------- |
| 通知（プッシュ通知） | 毎日のリマインド・タイマー終了通知 | **任意**（拒否しても本アプリは利用可能） |


本アプリは位置情報、連絡先、カメラ、マイク、写真ライブラリ、健康データ、広告 ID（IDFA / AAID）には**一切アクセスしません**。

## 2. 第三者サービスへのデータ送信

### 2-1. Firebase（Google LLC）

本アプリは Google が提供する Firebase を利用します。

- **Firebase Authentication**: 匿名 UID の発行・管理
- **Cloud Firestore**: 上記アプリ内データの保存（東京リージョン）
- **Cloud Functions for Firebase**: OpenAI API へのプロキシ（東京リージョン）
- **Firebase App Check**: 不正クライアントからの保護

Firebase のプライバシーポリシー: [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

### 2-2. OpenAI（OpenAI, L.L.C.）

アクション提案・パートナーメッセージ生成のため、以下の情報を **OpenAI の API（米国サーバー）** に送信します。


| 送信されるもの                 | 送信されないもの   |
| ----------------------- | ---------- |
| 現在のブレインスコア（0〜100 の数値）   | 匿名 UID     |
| 選択中のパートナータイプ            | 端末情報       |
| 言語設定                    | 個人を特定できる情報 |
| 直近のアクション履歴（最大3件のタイトル要約） | スコア履歴の詳細   |


OpenAI に送信したデータは、OpenAI のポリシーに従い **モデル学習には使用されません**。

OpenAI の API データ利用ポリシー: [https://platform.openai.com/docs/models/how-we-use-your-data](https://platform.openai.com/docs/models/how-we-use-your-data)

### 2-3. Expo（Expo, Inc.）

本アプリは Expo フレームワークでビルドされていますが、**ユーザーデータを Expo のサーバーに送信することはありません**。プッシュ通知トークンを取得する場合のみ、Expo のプッシュ通知サービスを経由します（トークン自体は端末ローカルに保持されます）。

## 3. データの保存期間と削除

### 3-1. 保存期間

ユーザーが本アプリをアンインストールした後も、Firestore に保存されたデータは残り続けます。これは、再インストール時に同一の匿名 UID でデータを復元するためです。

### 3-2. 削除方法

ユーザーは以下のいずれかでデータを削除できます。

1. **アプリ内: 設定 → データをリセット**
  - Firestore に保存されたプロフィール・スコア履歴がすべて削除されます
  - 端末ローカルの設定もすべて消去されます
2. **問い合わせ**（特殊な事情がある場合）
  - 下記の連絡先までメールにて削除依頼をしてください
  - 匿名 UID（設定画面で確認可能）を添えると確実です

## 6. ユーザーの権利

EU 一般データ保護規則（GDPR）、カリフォルニア州消費者プライバシー法（CCPA）、その他適用される法令に基づき、ユーザーは以下の権利を有します。

- 保存されている自分のデータへのアクセス権
- データの訂正を要求する権利
- データの削除を要求する権利（「忘れられる権利」）
- データ処理に異議を申し立てる権利
- データポータビリティの権利

これらの権利を行使したい場合は、下記の連絡先までお問い合わせください。

## 7. セキュリティ

- 通信はすべて **HTTPS / TLS で暗号化** されます
- Firestore Rules により、ユーザーは自分のデータのみアクセス可能です
- OpenAI API キーは Firebase Functions のサーバー側 Secret Manager で管理され、クライアントに公開されません
- Firebase App Check により、正規のアプリからのリクエストのみ受け付けます

ただし、インターネット上のデータ送信に絶対的な安全性はないことをご了承ください。
