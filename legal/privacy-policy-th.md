brain-detox (ต่อไปนี้เรียกว่า "แอป") เคารพความเป็นส่วนตัวของผู้ใช้ และจัดเก็บข้อมูลส่วนบุคคลให้น้อยที่สุดเท่าที่จำเป็น นโยบายนี้อธิบายประเภทของข้อมูลที่แอปจัดการ วัตถุประสงค์ สถานที่จัดเก็บ และสิทธิ์ของผู้ใช้

## 1. ข้อมูลที่เราเก็บ

### 1-1. รหัสประจำตัวแบบไม่ระบุตัวตน

แอปใช้ **Firebase Authentication แบบ Anonymous Sign-In** เพื่อออกรหัสประจำตัว (UID) ที่ไม่ระบุตัวตนสำหรับผู้ใช้แต่ละราย รหัสนี้ **ไม่เชื่อมโยงกับชื่อ อีเมล หมายเลขโทรศัพท์ หรือข้อมูลส่วนบุคคลใด ๆ**

### 1-2. ข้อมูลภายในแอป

ข้อมูลต่อไปนี้จะถูกจัดเก็บที่เซิร์ฟเวอร์ของเรา (Google Cloud Firestore, ภูมิภาคโตเกียว `asia-northeast1`):


| ข้อมูล                 | รายละเอียด                                                           | วัตถุประสงค์                             |
| ---------------------- | -------------------------------------------------------------------- | ---------------------------------------- |
| การตั้งค่าพาร์ทเนอร์   | ประเภทพาร์ทเนอร์ที่เลือก (teacher / counselor / scientist / trainer) | ปรับข้อเสนอกิจกรรมให้เหมาะสม             |
| คะแนนสมอง              | ค่าตัวเลข 0–100                                                      | ฟังก์ชันหลักของแอป                       |
| ภาษา                   | `ja` / `en` / `th`                                                   | การแสดงผล UI                             |
| ประวัติคะแนน           | บันทึกการทำกิจกรรมและการเทสต์ (สูงสุด 60 รายการ)                     | แสดงประวัติ คำนวณสตรีค                   |
| การตั้งค่าการแจ้งเตือน | เปิด/ปิด เวลาเตือน                                                   | **จัดเก็บเฉพาะบนอุปกรณ์** ไม่ส่งไปคลาวด์ |
| สถานะเหรียญตรา         | **จัดเก็บเฉพาะบนอุปกรณ์** ไม่ส่งไปคลาวด์                             | แสดงความสำเร็จ                           |
| ตัวนับ Rate Limit      | จำนวนการเรียก API ต่อวัน                                             | ป้องกันการใช้งานผิดปกติ ควบคุมต้นทุน     |


### 1-3. สิทธิ์อุปกรณ์


| สิทธิ์              | การใช้งาน              | บังคับ / เลือกได้                    |
| ------------------- | ---------------------- | ------------------------------------ |
| การแจ้งเตือน (Push) | เตือนรายวัน, จับเวลาจบ | **เลือกได้** (แอปใช้งานได้แม้ปฏิเสธ) |


แอป **ไม่เข้าถึง** ตำแหน่งที่ตั้ง รายชื่อผู้ติดต่อ กล้อง ไมโครโฟน คลังภาพ ข้อมูลสุขภาพ หรือรหัสโฆษณา (IDFA / AAID)

## 2. บริการบุคคลที่สาม

### 2-1. Firebase (Google LLC)

แอปใช้บริการของ Firebase:

- **Firebase Authentication**: ออก UID แบบไม่ระบุตัวตน
- **Cloud Firestore**: จัดเก็บข้อมูลข้างต้น (ภูมิภาคโตเกียว)
- **Cloud Functions for Firebase**: พร็อกซีสำหรับ OpenAI API (ภูมิภาคโตเกียว)
- **Firebase App Check**: ป้องกันไคลเอนต์ที่ไม่ได้รับอนุญาต

นโยบายความเป็นส่วนตัวของ Firebase: [https://firebase.google.com/support/privacy](https://firebase.google.com/support/privacy)

### 2-2. OpenAI (OpenAI, L.L.C.)

เพื่อสร้างข้อเสนอกิจกรรมและข้อความจากพาร์ทเนอร์ ข้อมูลต่อไปนี้จะถูกส่งไปยัง **OpenAI API (เซิร์ฟเวอร์สหรัฐฯ)**:


| ส่งไป                                        | ไม่ส่งไป                 |
| -------------------------------------------- | ------------------------ |
| คะแนนสมองปัจจุบัน (0–100)                    | UID ไม่ระบุตัวตน         |
| ประเภทพาร์ทเนอร์ที่เลือก                     | ข้อมูลอุปกรณ์            |
| ภาษา                                         | ข้อมูลที่ระบุตัวบุคคลได้ |
| สรุปกิจกรรมล่าสุด (สูงสุด 3 ชื่อ ตัดให้สั้น) | ประวัติคะแนนละเอียด      |


ตามนโยบายของ OpenAI **ข้อมูลที่ส่งผ่าน API จะไม่ถูกใช้ในการเทรนโมเดล**

นโยบายการใช้ข้อมูล OpenAI API: [https://platform.openai.com/docs/models/how-we-use-your-data](https://platform.openai.com/docs/models/how-we-use-your-data)

### 2-3. Expo (Expo, Inc.)

แอปสร้างด้วย Expo framework แต่ **ไม่มีการส่งข้อมูลผู้ใช้ไปยังเซิร์ฟเวอร์ของ Expo** โทเค็นการแจ้งเตือนอาจผ่านบริการของ Expo เท่านั้น โดยโทเค็นเก็บไว้ที่อุปกรณ์ของคุณ

## 3. การเก็บรักษาและการลบข้อมูล

### 3-1. ระยะเวลาการเก็บรักษา

หลังจากที่คุณถอนการติดตั้งแอป ข้อมูลใน Firestore จะยังคงอยู่ เพื่อให้สามารถกู้คืน UID แบบไม่ระบุตัวตนเดิมเมื่อติดตั้งใหม่

### 3-2. วิธีการลบ

ผู้ใช้สามารถลบข้อมูลได้ด้วยวิธีต่อไปนี้:

1. **ในแอป: ตั้งค่า → รีเซ็ตข้อมูล**
  - ลบโปรไฟล์และประวัติคะแนนที่จัดเก็บใน Firestore
  - ล้างการตั้งค่าทั้งหมดบนอุปกรณ์
2. **ติดต่อเรา** (กรณีพิเศษ)
  - ส่งอีเมลตามที่อยู่ด้านล่าง
  - ระบุ UID ไม่ระบุตัวตน (ดูได้ในหน้าตั้งค่า) เพื่อความรวดเร็ว

## 6. สิทธิ์ของผู้ใช้

ภายใต้ EU GDPR, California CCPA และกฎหมายที่บังคับใช้อื่น ๆ ผู้ใช้มีสิทธิ์ต่อไปนี้:

- สิทธิ์เข้าถึงข้อมูลของตนเอง
- สิทธิ์ขอแก้ไข
- สิทธิ์ขอลบ ("สิทธิ์ที่จะถูกลืม")
- สิทธิ์คัดค้านการประมวลผล
- สิทธิ์ในการเคลื่อนย้ายข้อมูล

หากต้องการใช้สิทธิ์เหล่านี้ กรุณาติดต่อตามที่อยู่ด้านล่าง

## 7. ความปลอดภัย

- การสื่อสารทั้งหมดเข้ารหัสด้วย **HTTPS / TLS**
- Firestore Rules ทำให้ผู้ใช้แต่ละรายเข้าถึงได้เฉพาะข้อมูลของตน
- คีย์ OpenAI API เก็บไว้ใน Firebase Functions Secret Manager และไม่เปิดเผยต่อไคลเอนต์
- Firebase App Check รับคำขอจากแอปที่ถูกต้องเท่านั้น

อย่างไรก็ตาม การส่งข้อมูลทางอินเทอร์เน็ตไม่มีความปลอดภัยที่สมบูรณ์แบบ
